# Program ini menjalankan fungsi untuk menghitung total biaya pengiriman barang :
1.Berat benda (Kilogram)

2.Jarak Pengiriman (Kilometer)

3.Volume Barang (cm³, Dengan rumus = panjang X lebar X tinggi)

Biaya Pengiriman Barang Ditentukan Oleh :

-Jika jarak ≤ 10 km, maka biaya per kilogram = Rp 4.250

-Jika jarak > 10 km, maka biaya per kilogram = Rp 6.000

-Jika volume > 100 cm³, maka ditambah biaya Rp 50.000
